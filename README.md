[mailto](mailto:aajayiadeyemi@gmail.com)

## Git pull
` git pull https://github.com/acidicyemi/APPointment.git`

## Composer Install
`composer install`

## NPM install
`npm i`

## Set up database.sqlite
Edit your env file
------------------
`DB_CONNECTION=sqlite`
Create a databas.sqlite at database

## Run migrations
`php artisan migrate`
